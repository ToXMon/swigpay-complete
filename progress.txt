## Codebase Patterns
- `@sqds/multisig` permission helpers are accessed via `multisig.types.*`, not top-level exports.
- `multisig.rpc.multisigCreateV2` needs `treasury` from `ProgramConfig.treasury` and `rentCollector` explicitly set.
- `multisigAddSpendingLimit` uses `bigint` amount, while `spendingLimitUse` uses `number` amount.
- USDC/Solana defaults now live in `packages/agent-wallet/src/config.ts` and MCP server defaults live in `apps/mcp-server/src/config.ts`.
- Dashboard typecheck requires `apps/dashboard/tsconfig.json` and `apps/dashboard/next-env.d.ts`.
